# Release Notes / 출시노트

## Version 1.1.0

```
<ko-KR>
✨ 새로운 기능
- 🗺️ 하루 이동 경로를 지도에서 확인할 수 있는 위치 추적 기능 추가
- 🎬 이동 경로 애니메이션 영상 재생 및 녹화 기능
- 📝 날짜별 일기 작성 및 이모지 선택 기능
- 📅 캘린더에서 일기가 있는 날짜를 이모지로 표시
- 🔍 일기 검색 기능 추가
- 📍 위치 기록 상세 보기 화면 추가
- 🌐 한국어 / 영어 다국어 지원
- 💾 로컬 백업 및 복원 기능
- ⏱️ 위치 추적 간격 설정 지원 (1분 ~ 30분)
- 📱 백그라운드 위치 추적 지원
</ko-KR>
```
