## 1. 개요

**"발자국 일기"** 는 사용자의 일상 경로를 기록하고 위치 기반 일기를 작성할 수 있는 모바일 앱입니다.

**본 앱은 서버를 운영하지 않으며, 모든 데이터는 사용자의 기기에만 저장됩니다.**

## 3. 핵심 개인정보 처리 원칙

### 서버를 사용하지 않습니다

본 앱은 **별도의 서버를 운영하지 않습니다.** 사용자가 작성한 모든 데이터는 사용자의 기기 내부에만 저장되며, 개발자를 포함한 그 누구도 사용자의 데이터에 접근할 수 없습니다.

| 데이터 유형 | 저장 위치 | 외부 전송 |
|------------|----------|----------|
| 일기 내용 | 기기 내 SQLite DB | ❌ 없음 |
| 위치 기록 | 기기 내 SQLite DB | ❌ 없음 |
| 백업 파일 | 기기 내 저장소 | ❌ 없음 |
| 녹화 영상 | 기기 갤러리 | ❌ 없음 |
| 앱 설정 | 기기 내 저장소 | ❌ 없음 |

## 4. 앱 권한 및 사용 목적

### 4.1 위치 정보 (필수)

- **권한**: GPS 위치 접근, 백그라운드 위치 접근
- **목적**: 사용자의 이동 경로를 기록하여 일기와 함께 지도에 표시
- **저장**: 기기 내부에만 저장, 외부 전송 없음
- **제어**: 앱 설정에서 위치 추적을 언제든 끄거나 삭제 가능

### 4.2 저장소 접근 (선택)

- **권한**: 파일 읽기/쓰기
- **목적**: 일기 데이터 백업/복원, 경로 영상 저장
- **저장**: 사용자가 지정한 기기 내 폴더에만 저장

### 4.3 화면 녹화 (선택)

- **권한**: 화면 녹화 (오디오 녹음 없음)
- **목적**: 경로 애니메이션을 영상으로 저장
- **저장**: 기기 갤러리에만 저장, 외부 전송 없음

## 5. 제3자 서비스

### 5.1 Google AdMob (광고)

본 앱은 무료 서비스 제공을 위해 Google AdMob 배너 광고를 표시합니다.

- **수집 주체**: Google LLC
- **수집 데이터**: 광고 표시를 위한 기기 식별자, 광고 상호작용 정보
- **사용자 일기/위치 데이터**: AdMob에 전송되지 않음
- **개인정보처리방침**: [Google 개인정보처리방침](https://policies.google.com/privacy)

### 5.2 Firebase Analytics

- **상태**: 비활성화됨 (IS_ANALYTICS_ENABLED: false)
- 본 앱은 Firebase Analytics를 사용하지 않습니다.
- 사용자 행동 분석 데이터를 수집하지 않습니다.

## 7. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

| 권리 | 방법 |
|------|------|
| **데이터 확인** | 앱 내에서 모든 일기 및 위치 기록 확인 가능 |
| **데이터 삭제** | 앱 설정에서 개별 일기 또는 전체 데이터 삭제 |
| **데이터 내보내기** | 설정 > 백업에서 JSON 파일로 내보내기 |
| **권한 철회** | 기기 설정에서 위치/저장소 권한 해제 |
| **앱 삭제** | 앱 삭제 시 모든 데이터가 기기에서 완전히 삭제됨 |

## 8. 데이터 보안

- 모든 데이터는 사용자 기기 내부에만 저장됩니다.
- 네트워크를 통한 데이터 전송이 없으므로 데이터 유출 위험이 없습니다.
- 앱 삭제 시 모든 데이터가 영구적으로 삭제됩니다.
- 백업 파일은 사용자가 직접 관리하며, 기기 외부로 전송되지 않습니다.
