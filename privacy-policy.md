## 1. 개요

**"발자국 일기"(이하 "앱")** 는 사용자의 일상적인 경로를 추적하고 위치 기반 일기 작성을 돕는 모바일 애플리케이션입니다. 저희는 사용자의 개인정보를 보호하고 투명하게 처리하는 것을 최우선으로 삼고 있습니다.

본 개인정보 처리방침은 앱 사용 중 수집되는 정보와 그 사용 방식에 대해 설명합니다.

## 3. 수집하는 개인정보

### 3.1 위치 정보 (GPS)

**수집 항목:**
- 사용자의 현재 위치 좌표 (위도, 경도)
- 일일 경로 데이터 (방문 장소 및 이동 경로)

**수집 목적:**
- 사용자의 일일 경로 추적 및 지도 표시
- 위치 기반 일기 작성 지원

**수집 방식:**
- 명시적 권한 요청 후 필요한 경우에만 수집
- 사용자가 앱 내 설정에서 언제든지 거부 또는 중지 가능

### 3.2 저장소 접근 권한

**수집 항목:**
- 디바이스 저장소의 파일 읽기/쓰기 권한

**수집 목적:**
- 일기 데이터의 백업 및 복구
- 경로 애니메이션 비디오 저장
- 사용자 생성 콘텐츠 관리

**수집 방식:**
- 명시적 권한 요청 후 필요한 경우에만 접근
- 사용자가 앱 내 설정에서 언제든지 거부 또는 중지 가능

### 3.3 화면 기록 (Screen Recording)

**수집 항목:**
- 경로 기반의 화면 녹화 (오디오 없음)
- 경로 애니메이션 비디오 데이터

**수집 목적:**
- 사용자의 일일 경로를 시각적 애니메이션 비디오로 변환
- 경로 시각화 및 기억 보관

**수집 방식:**
- 사용자의 명시적 요청 시에만 실행
- 오디오는 녹화되지 않음
- 생성된 비디오는 디바이스에만 저장됨

### 3.4 Firebase Analytics

**수집 항목:**
- 앱 사용 패턴 (시작, 종료 시간)
- 기능 사용 통계
- 기기 정보 (기기 종류, OS 버전, 앱 버전)
- 에러 및 성능 데이터

**수집 목적:**
- 앱 사용성 분석 및 개선
- 기술적 성능 모니터링
- 사용자 경험 향상을 위한 통계 수집

**수집 방식:**
- 자동으로 수집되는 활용 데이터
- Firebase Analytics의 데이터 수집 정책에 따름

## 5. 정보의 저장 및 보안

### 5.1 저장소

**모든 사용자 데이터는 사용자의 디바이스에만 로컬로 저장됩니다.**

- 위치 데이터: 디바이스 내부 데이터베이스
- 일기 기록: 디바이스 저장소
- 경로 애니메이션 비디오: 디바이스 저장소

클라우드 서버에는 저장되지 않으며, 디바이스 내에서만 관리됩니다.

### 5.2 데이터 보안

- SQLite 데이터베이스를 통한 구조화된 저장
- 데이터 접근은 앱 내에서만 가능
- 시스템 권한에 따른 보호

## 7. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

### 7.1 데이터 접근권

- 언제든지 앱 내에서 자신의 위치 데이터 및 일기 기록 확인 가능

### 7.2 데이터 삭제권

**사용자는 언제든지 모든 데이터를 삭제할 수 있습니다:**

- 앱 내 설정 메뉴에서 "데이터 초기화" 또는 "모든 데이터 삭제" 옵션 이용
- 앱 제거 시 디바이스의 모든 저장 데이터 삭제
- 개발사 연락처로 요청하여 데이터 삭제 지원 받기 가능

### 7.3 권한 거부권

- OS 설정에서 앱의 위치, 저장소 접근 권한 언제든지 거부 가능
- 앱 내 설정에서 특정 기능 비활성화 가능

## 11. 권장사항 및 주의사항

### 11.1 앱 권한 관리

사용자는 다음과 같이 권한을 관리할 수 있습니다:

**Android:**
- 설정 > 앱 > 발자국 일기 > 권한
